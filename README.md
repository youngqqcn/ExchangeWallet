- AM和WM使用 qt creator 4.5.0   base on  Qt 5.10.0(MSVC2015, 32bit) 
- AM和WM依赖`libs`下一些库(dll, lib), 这些库使用vs2017(带有vs2015 32bit编译套件)进行
- python服务使用的是 python3.7.4
- `tools\GeneratePublicKey\src` 使用的是 python2.7.
